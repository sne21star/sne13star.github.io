Course Description

This course is an introductory programming course that teaches Python. But in addition to that, it provides an introduction to topics in data science. Topics covered include: Basics of git Regular expressions and text processing Python basics Python data structures and libraries Basic object-oriented programming Basic data visualization Sampling, estimation, hypothesis testing Regression analyses Classification and clustering Basic neural networks https://catalog.purdue.edu/preview_course_nopop.php?catoid=13&coid=133918
